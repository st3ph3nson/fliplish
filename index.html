Pig Latin Suffix Translator

Standard Pig Latin core + a selectable ending (-ek / -eg / -en / -us / random). Punctuation-safe, case-preserving.

Rules: vowel-start word → word + ending, consonant cluster moves (e.g. consonant → onsonantc + ending, cluster → ustercl + ending).
Examples (ending = -us): apple → appleus, cluster → usterclus
